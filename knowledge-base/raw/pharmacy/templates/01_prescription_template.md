# Образец рецепта на отпуск препарата

## Назначение
Этот образец используется как эталон для проверки, какие реквизиты должны быть заполнены в рецепте перед отпуском препарата.

## Обязательные поля
- фамилия и имя пациента;
- дата выписки;
- наименование препарата;
- дозировка;
- лекарственная форма;
- количество;
- подпись назначающего специалиста;
- отметка медицинской организации, если она предусмотрена правилами.

## Правила заполнения
1. Все поля должны быть читаемыми и однозначными.
2. Не допускаются исправления без подтверждения.
3. Наименование препарата должно совпадать с назначением.
4. Количество и дозировка должны быть указаны явно.
5. Рецепт должен позволять проверить срок действия и легитимность назначения.

## Для чего нужен шаблон
Шаблон позволяет сравнить реальный рецепт с эталонной структурой и быстро понять, чего не хватает для отпуска препарата.
