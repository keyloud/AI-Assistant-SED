# Образец отчета по отпуску или проверке препарата

## Назначение
Этот шаблон нужен для оформления краткого отчета по выполненной проверке, отказу в отпуске или завершенной операции.

## Структура отчета
- дата и время события;
- сотрудник, выполнивший проверку;
- наименование препарата или документа;
- результат проверки;
- основание для решения;
- замечания или пояснения.

## Пример формулировок
- отпуск разрешен по результатам проверки;
- отпуск запрещен из-за отсутствия обязательных реквизитов;
- документ передан на дополнительную проверку;
- товар изъят из оборота и направлен на списание.

## Требования к оформлению
1. Отчет должен быть кратким и однозначным.
2. Причина решения должна быть записана без двусмысленности.
3. Если есть спорный случай, указывается ответственное лицо.
4. При необходимости добавляется ссылка на внутреннее основание или регламент.
