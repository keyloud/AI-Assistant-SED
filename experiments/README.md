# experiments/

Содержит всё необходимое для оценки метода (Фаза 6):

- `eval-dataset/` — 20 вопросов с эталонными ответами по всем 4 сценариям
- `evaluation/`   — скрипты A/B сравнения (с контекстом vs без)
- `notebooks/`    — Jupyter notebooks с результатами экспериментов
- `logs/`         — JSON-логи каждого pipeline-запуска (ExperimentLoggingStep)
